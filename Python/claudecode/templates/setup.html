Welcome to Medicine Inventory Manager

Setup required to get started

1

Notion Integration Required

This application requires Notion database setup to store your medicine inventory data.

2

Follow Setup Guide

Please follow the NOTION_SETUP.md file to configure your Notion databases.

3

Configure Environment

Update your .env file with Notion integration token and database IDs.

Quick Setup Steps:

  1. Create a Notion integration at notion.so/my-integrations
  2. Create three databases: Patients, Medicines, Purchases
  3. Share each database with your integration
  4. Copy database IDs and integration token to .env file
  5. Restart the application
